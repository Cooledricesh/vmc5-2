#### 1. 대시보드 페이지

## 📈 기술 스택 일관성 검증

### ✅ 일관되게 사용된 기술
- **Frontend**: Next.js 14 (App Router), React 19
- **상태 관리**: Context + useReducer (Level 3 복잡도)
- **서버 상태**: @tanstack/react-query
- **UI**: Shadcn-ui, Tailwind CSS
- **Backend**: Hono.js
- **Database**: Supabase (PostgreSQL)
- **인증**: Clerk SDK
- **결제**: 토스페이먼츠
- **AI**: Google Gemini API
- **검증**: Zod

### ✅ 코딩 표준 준수

- 에러 코드 표준화
